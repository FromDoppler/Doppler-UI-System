Carousel Component

Al copiar el html del carousel cambiar id de cada slide, siempre dejando la terminación _dpslide.
Ejemplo: id="primero_dpslide", id="segundo_dpslide" , etc.

Poner misma cantidad de inputs que de slides. A cada input pornerle un id y value distinto, y un mismo name para todos los inputs del mismo carousel.
Ejemplo: para el primer input de un carousel id="primero" value="primero" name="carousel1" . Para el segundo input id="segundo" value="segundo" name="carousel1"
